# Proyecto de Chat en Tiempo Real

Este es un proyecto de chat en tiempo real desarrollado con Node.js, Express y Socket.io.

## Tecnologías Utilizadas

- **Node.js**: Entorno de ejecución para JavaScript en el servidor.
- **Express**: Framework web para Node.js.
- **Socket.io**: Biblioteca para aplicaciones web en tiempo real.

## Instalación

1. Clona el repositorio:
    ```bash
    git clone https://github.com/tu_usuario/chat_tiempo_real.git
    ```
2. Navega al directorio del proyecto:
    ```bash
    cd chat_tiempo_real
    ```
3. Instala las dependencias:
    ```bash
    npm install
    ```

## Uso

1. Inicia el servidor:
    ```bash
    npm start
    ```
2. Abre tu navegador y navega a `http://localhost:3000`.

## Funcionalidades

- Envío y recepción de mensajes en tiempo real.
- Notificaciones de usuarios conectados y desconectados.
- Interfaz de usuario simple y amigable.
